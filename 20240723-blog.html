Maintainability

23 July 2024

Even as a newbie, one of the main things I want to get good at is writing clean, simple code.

One thing I have noticed when inspecting websites I like is that lots of them define their colour scheme at the start of their CSS document, and then refer to them by name throughout the style sheet.

This is such a great idea because it means that if you suddenly have a change of heart and want to completely change your colour scheme, you can change the colour values in one place, hit save, and the changes are applied to everywhere. What a time saver!

So that's what I've done with this blog. I have started my style.css as follows:

:root {
    --col-brand: #878ca5;
    --col-light: #fff8e8;
    --col-dark: #41496d;
    --col-light-accent: #ffd369;
    --col-dark-accent: #9a64da;
    --col-dark-accent-2: #6a7eda;
}

Even as I write this entry, I'm thinking of changing my original colour scheme: I was going for 'sleek and professional,' but instead it feels a bit too much like '70 year old man's first website', so I'll definitely be taking advantage of this new trick!
